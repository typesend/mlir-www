42 /// Printer hook for custom directive in assemblyFormat.
43 ///
44 /// custom<DynamicIndexList>($values, $integers)
45 ///
46 /// where `values` is of ODS type `Variadic<Index>` and `integers` is of ODS
47 /// type `I64ArrayAttr`. Prints a list with either (1) the static integer value
48 /// in `integers` is `dynVal` or (2) the next value otherwise. This allows
49 /// idiomatic printing of mixed value and integer attributes in a list. E.g.
50 /// `[%arg0, 7, 42, %arg42]`.
51 void printDynamicIndexList(
52  OpAsmPrinter &printer, Operation *op, OperandRange values,
53  ArrayRef<int64_t> integers,
54  AsmParser::Delimiter delimiter = AsmParser::Delimiter::Square);

56 /// Pasrer hook for custom directive in assemblyFormat.
57 ///
58 /// custom<DynamicIndexList>($values, $integers)
59 ///
60 /// where `values` is of ODS type `Variadic<Index>` and `integers` is of ODS
61 /// type `I64ArrayAttr`. Parse a mixed list with either (1) static integer
62 /// values or (2) SSA values. Fill `integers` with the integer ArrayAttr, where
63 /// `dynVal` encodes the position of SSA values. Add the parsed SSA values
64 /// to `values` in-order.
65 //
66 /// E.g. after parsing "[%arg0, 7, 42, %arg42]":
67 /// 1. `result` is filled with the i64 ArrayAttr "[`dynVal`, 7, 42, `dynVal`]"
68 /// 2. `ssa` is filled with "[%arg0, %arg1]".
69 ParseResult parseDynamicIndexList(
70  OpAsmParser &parser,
71  SmallVectorImpl<OpAsmParser::UnresolvedOperand> &values,
72  DenseI64ArrayAttr &integers,
73  AsmParser::Delimiter delimiter = AsmParser::Delimiter::Square);
