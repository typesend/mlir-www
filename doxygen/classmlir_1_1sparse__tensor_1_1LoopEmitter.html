void LoopEmitter::initialize ( ValueRange  tensors,
StringAttr  loopTag = nullptr,
bool  hasOutput = false,
bool  isSparseOut = false,
ArrayRef< LoopId >  topSort = {},
DependentLvlGetter  getter = nullptr 
)

Takes an array of input tensors, which the generated loops will iterate over.

Each tensor is given a TensorId (numerically equal to the position of that tensor Value in the array). Setting isSparseOut indicates that the sparse output tensor is empty, so the loop emitter will generate loops over it according to the level-sizes. The topSort array specifies the actual order in which loops are generated, thus providing a mapping from LoopOrd to LoopId.
