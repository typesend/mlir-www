static

Return the untiled producer whose slice is used in a tiled consumer.

The method traverses the tile loop nest (loops) if needed, and returns the iter_args of the outer most that is encountered. Traversing the iter_args indicates that this is a destination operand of the consumer. If there was no loop traversal needed, the second value of the returned tuple is empty.

}

For a value to be yielded (yieldedValue) from within a loop nest loops, construct the destructive update pattern that inserts the yielded value into a destination tensor provided by initValue at offset tileOffsets and size tileSizes.

For example,

scf.for %iv0 = ... {
%0 = tiled_op
}

is transformed to

scf.for %iv0 = ... iter_args(%arg = %0) {
%1 = tensor.extract_slice %arg
%2 = tiled_op
%3 = tensor.insert_slice %2 into %arg
scf.yield %3
}
