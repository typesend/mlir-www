36 /// Helper type for functions that generate ids for the mapping of a
37 /// scf.forall.
38 struct IdBuilderResult {

49 /// Common gpu id builder type, allows the configuration of lowering for various
50 /// mapping schemes. Takes:
51 /// - A rewriter with insertion point set before the forall op to rewrite.
52 /// - The loc of the forall op to rewrite.
53 /// - A list of positive integers carrying the mapping sizes for the current
54 /// forall op to rewrite.

56  std::function<IdBuilderResult(RewriterBase &, Location, ArrayRef<int64_t>)>;

58 /// Helper struct for configuring the rewrite of mapped scf.forall ops to

63  mappingAttributes(), idBuilder() {}

65  /// List of OpFoldResult carrying the multi-dimensional number of

73 
74  /// The mapping attributes targeted by this generator.

93 /// Search `scf.forall` ops nested under `target` and map each such op to an
94 /// explicit GPU implementation along `availableMappingSizes`.
95 /// The mapping is one-to-one and the induction variables of `scf.forall` are
96 /// rewritten to gpuIdBuilder.idBuilder according to the

107 /// Search `scf.forall` ops nested under `target` and map each such op to an
108 /// explicit GPU implementation along blockDims and warpDims.
109 /// The mapping is one-to-one and the induction variables of `scf.forall` are
110 /// rewritten to threads and warps ids according to the mapping attribute.
111 ///
112 /// Dynamic, `scf.forall` trip counts are currently not supported.
113 /// Dynamic `blockDims` or `warpDims` or `linearDims` sizes are currently not

116 /// `warpDims` is expected to be empty or of size 3.
117 ///

119 /// beginning of the `target` body block and dominate all other blocks.
120 DiagnosedSilenceableFailure mapNestedForallToThreadsImpl(

122  Operation *target, ArrayRef<int64_t> blockDimsOfr,
