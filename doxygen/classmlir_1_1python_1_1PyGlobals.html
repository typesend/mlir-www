◆ loadDialectModule()

void PyGlobals::loadDialectModule ( llvm::StringRef  dialectNamespace)

Loads a python module corresponding to the given dialect namespace.

No-ops if the module has already been loaded or is not found. Raises an error on any evaluation issues. Note that this returns void because it is expected that the module contains calls to decorators and helpers that register the salient entities.
