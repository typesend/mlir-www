Detailed Description

template<typename OpTy>
class mlir::transform::TransformWithPatternsOpTrait< OpTy >

Trait implementing the applyToOne function required by TransformEachOpTrait by greedily applying a set of patterns to each target payload operation.

This requires the transform operation to implement TransformEachOpTrait and to provide the following method:

  • void populatePatterns(RewritePatternSet &) that populates the given object with the patterns to apply. This is an instance method that can depend on the transform operation attributes.

The payload operation is expected to have the IsolatedFromAboveTrait, which is a requirement of the pattern rewriter. If it does not, or if pattern application fails, the transform fails definitively as the rewriter will have likely left the payload IR in some intermediate state that precludes further transformation.
