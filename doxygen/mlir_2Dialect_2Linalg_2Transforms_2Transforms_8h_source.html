385 /// Mechanically hoist padding operations on tensors by `numLoops` into a new,
386 /// generally larger tensor. This achieves packing of multiple padding ops into
387 /// a larger tensor. On success, `opToHoist` is replaced by the cloned version
388 /// in the packing loop so the caller can continue reasoning about the padding
389 /// operation. If `transposeVector` is non-empty, hoist padding introduces a
390 /// GenericOp to transpose the padded tensor before inserting it into the packed
391 /// tensor. A `transposeVector` can change the storage order of the padded
392 /// tensor but does not change the order of the pack or compute loops.
393 ///
394 /// TODO: In the future, we should consider rewriting as a tensor.pack after
395 /// hoisting since this abstraction is now available.
396 ///
397 /// Example in pseudo-mlir:
398 /// =======================
399 ///
400 /// If hoistPaddingOnTensors is called with `nLoops` = 2 on the following IR.
401 /// ```
402 /// scf.for (%i, %j, %k)
403 /// %st0 = tensor.extract_slice f(%i, %k) : ... to tensor<?x?xf32>
404 /// %0 = tensor.pad %st0 low[0, 0] high[...] {
405 /// ^bb0( ... ):
406 /// linalg.yield %pad
407 /// } : tensor<?x?xf32> to tensor<4x8xf32>
408 /// compute(%0)
409 /// ```
410 ///
411 /// IR resembling the following is produced:
412 ///
413 /// ```
414 /// scf.for (%i) {
415 /// %packed_init = tensor.empty range(%j) : tensor<?x4x8xf32>
416 /// %packed = scf.for (%k) iter_args(%p : %packed_init) {
417 /// %st0 = tensor.extract_slice f(%i, %k) : ... to tensor<?x?xf32>
418 /// %0 = tensor.pad %st0 low[0, 0] high[...] {
419 /// ^bb0( ... ):
420 /// linalg.yield %pad
421 /// } : tensor<?x?xf32> to tensor<4x8xf32>
422 /// %1 = tensor.insert_slice %0 ...
423 /// : tensor<4x8xf32> to tensor<?x4x8xf32>
424 /// scf.yield %1: tensor<?x4x8xf32>
425 /// } -> tensor<?x4x8xf32>
426 /// scf.for (%j, %k) {
427 /// %st0 = tensor.extract_slice %packed [%k, 0, 0][1, 4, 8][1, 1, 1] :
428 /// tensor<?x4x8xf32> to tensor<4x8xf32>
429 /// compute(%st0)
430 /// }
431 /// }
432 /// ```
